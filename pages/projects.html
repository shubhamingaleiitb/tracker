Q1: Implement a Transformer from Scratch Months 1–3

Not following a tutorial. Read the original paper, implement multi-head attention, positional encoding, the full encoder-decoder. Train on a small task. Profile, identify bottlenecks, optimize.

  • Read "Attention Is All You Need" with pen and paper Paper
  • Implement multi-head attention from the paper (no tutorial)
  • Implement positional encoding (sinusoidal)
  • Implement feedforward blocks with residual connections and layer norm

  • Profile with PyTorch profiler — where is time spent? Why? Profiler Guide

Q4: Your Own Idea Months 10–12

After 8+ months of reading papers and reimplementing ideas, pursue your own question. It doesn't need to be a breakthrough — it needs to show original thinking.

What "your own idea" might look like:

  • An experiment that tests a hypothesis you formed while reading papers
  • A combination of two techniques that haven't been combined
  • A rigorous benchmark that doesn't exist yet in your area of interest
  • An open-source tool that embodies an insight from your paper reading
  • Define your research question / project idea
  • Literature review — what's been done in this space?
  • Implement core experiment / prototype
  • Run experiments and collect results
  • Write a thorough blog post / technical report with honest discussion
  • Open-source the code with good documentation
